# Descrription

A project in pygame to re-create iconic Asteroids game

wasd - move
space - shoot


# Set-up

## Install UV

Linux
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Check the command for your system at https://docs.astral.sh/uv/getting-started/installation/

## Sync

```
uv sync
```

## Run

```
uv run main.py
```
